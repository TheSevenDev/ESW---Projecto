Approved(Nullable<Int32>, string) Apresenta a vista com as candidaturas aprovadas, dispostas numa tabela paginada e filtradas por um critério passado por argumento.
CancelApplication() Ação que cancela uma candidatura submetida. Neste caso, atualiza o estado da candidatura para 'Descartada'. Este método utiliza como chave primária da candidatura a cancelar os valores que vêm no request AJAX feito pelo browser, aquando clique no botão de cancelamento da candidatura.
CancelationReason(string, string) Regista o motivo para o cancelamento da candidatura. Recebe como argumentos a chave primária da candidatura a cancelar e o texto com o motivo.
Confirm(int) Ação que confirma uma candidatura e a coloca como mobilidade. Redirecciona o utilizador para vista da sua mobilidade corrente.
CreateSignedApplication(ApplicationViewModel) Cria o documento comprotavito da submissão de uma candidatura. Este documento contem a assinatura do estudante, que é feita no formulário de criação de uma candidatura.
Details(string) Retorna uma partial view com os detalhes de uma candidaturas.
Evaluate(int, Nullable<Boolean>, Nullable<Boolean>) Retorna a vista com o formulário para a avaliação de uma candidatura. Caso não tenha sido escolhido nenhum técnico responsável ou uma instituição de destino retorna a view e mostra os respetivos erros.
EvaluateApplication(ApplicationEvaluationViewModel) Submete a avaliação de uma candidatura. Muda o estado da candidatura para aceite ou rejeitada, consoante a avaliação. Por fim notifica o aluno e o técnico responsável pela mobilidade.
EvaluationDetails(string) Retorna uma partial view com os detalhes da avaliação de uma candidaturas.
FetchInterviews() Método que procura todas as marcações de entrevistas registadas na base de dados e converte-as para um ficheiro XML. Este ficheiro é guardado no servidor para mais tarde ser usado na mostragem das marcações de entrevistas.
GetConfirmedApplications(ClaimsPrincipal) Retorna o número de candidaturas confirmadas do utilizador autenticado.
GetCurrentUserID() Retorna a chave primária associada à conta do utilizador autenticado no momento.
GetEmail(int) Retorna o email de uma conta com base na chave primária da mesma.
GetIdByInstitution(string) Retorna a chave primária de uma instituição pelo nome.
GetNewApplicationID() Retorna a última candidatura submetida
GetNumberPendentApplications(ClaimsPrincipal) Retorna o número de candidaturas com avaliação pendente do utilizador autenticado.
GetStudentById(int) Retorna o estudante com base na chave primária da conta.
HasConfirmedApp() Verifica se um estudante já tem uma candidatura aceite.
Index(Nullable<Int32>, string) Retorna a página index das candidaturas que contém uma lista paginada das candidaturas submetidas pelos estudantes filtrada pelo critério passado como argumento.
Interviews(int) Carrega a página com a calendarização das entrevistas.
MyApplications() Retorna a view com as candidaturas efetuadas pelo estudante autenticado. Caso não esteja autenticado ou não seja estudante, o utilizador é redireccionado para uma view de erro.
New(int) Retorna a vista com o formulário para a submissão de uma nova candidatura. Caso o utilizador não esteja autenticado, ou não seja estudante, ou já tenha uma candidatura aprovada é retornada uma vista com um erro.
OpenSignedAppFile(int)
RegisterApplication(ApplicationViewModel) Submete um novo registo para uma candidatura a um programa de mobilidade. Redirecciona o utilizador para a página das aplicações submetidas pelo mesmo.
RegisterApplicationInstitutions(string, string, string) Regista as instituições escolhidas pelo estudante para a candidatura. Pode escolher 1, 2 ou 3 instituições em que a ordem da escolha indica uma ordem de preferência. Este método é usado pelo request AJAX aquando submissão do formulário.
RescheduleInterview(string) Retorna a vista para reagendar uma data para a entrevista de uma candidatura.
RescheduleInterview(InterviewViewModel) Reagenda uma entrevista para a candidatura.
